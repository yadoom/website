Raspberry Pi Light Show

Source: AdaFruit Learning System

The Raspberry Pi is a versatile little thing. Its a low cost, low profile computer that can fit in the palm of your hand. It has a series of General Purpose Input Output (GPIO) pins that can be used to communicate with external hardware.

I found a guide on Adafruit. The source above includes all the open source code for getting the RPi to communicate with strips of LEDs.

I was able to tweak the Python source to better fit my specific hardware and tastes. These include adjusting for the length and maximum brightness of my LED strip, cycling colors, adjusting the refresh rate to minimize delay. I also expiremented with a manual mode that can show static colors and different effects.

It can be controlled remotely via ssh. I've lost the spectrum footage but it looked basically identical to the Adafruit GIF in the card above.

Tower of Hanoi Simulation

Wikipedia

This is a fun puzzle that has a neat background. Despite its apparent origins with Édouard Lucas of France, there are legends surrounding the puzzle.

While it has a few different variations, the legend goes that some mysterious temple in a remote land houses a large room with three posts and 64 golden disks. In this room, mysterious people have been moving the disks for thousands of years following the rules of an ancient prophecy. Since the minimum number of moves to solve the Tower of Hanoi problem given n disks is 2n-1, if each move took 1 second, it would take 42 times the age of the universe for the mysterious people to be finished.

Using Java and JavaFX, I decided to make a simulator for this puzzle. You can specify the number of disks (up to 10) and it will solve the problem for you showing each step with animation.
